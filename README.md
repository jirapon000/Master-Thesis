# Multi-Agent System for Clinical Assessment (PHQ-8)

## 📖 Overview
This repository contains the implementation of a **Multi-Agent System (MAS)** designed to automate and enhance clinical assessment procedures, specifically focusing on the **Patient Health Questionnaire-8 (PHQ-8)** for depression screening.

## ✨ Key Features
* **Multi-Agent Architecture:** Autonomous agents assigned to specific roles (e.g., Question Agent, Participant Simulator, Scoring Agent) to model the clinical interaction.
* **PHQ-8 Implementation:** Full integration of the 8-item depression scale scoring logic and protocol.
* **Interaction Workflow:** Structured communication protocols between agents to simulate realistic assessment dialogues.
* **Data Logging:** Automated recording of agent interactions and scoring results for analysis.
